# Accesibilidad y Diseño Universal para el Aprendizaje

## Acciones base

- Materiales clave en texto descargable.
- Videos con subtítulos o transcripción.
- Alternativas equivalentes para audio, video, código y baja conectividad.
- Plantillas compatibles con lector de pantalla.
- Rúbricas claras antes de entregar.

## Aplicación por módulo

| Módulo | Ajuste recomendado |
|---|---|
| Inducción | Ofrecer política en formato texto descargable y lectura guiada. |
| Fundamentos | Ofrecer ejemplos textuales además de demostración en vivo. |
| Prompting y verificación | Permitir bitácora en formato tabla, audio transcrito o documento narrativo. |
| Estudio y escritura | Permitir lectura con audio, resumen docente o versión de lectura fácil cuando corresponda. |
| Datos y código | Ruta alternativa con hoja de cálculo. |
| Comunicación y carrera | Permitir pitch oral, escrito o grabado con subtítulos. |
| Multimodalidad y proyecto | Aceptar formatos equivalentes de producto cuando cumplan resultados. |
